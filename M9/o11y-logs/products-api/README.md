# Products API

This is a simple Node.js application written in TypeScript that provides a products API.

## Installation

To install the dependencies, run the following command:

```bash
npm install
```

## Available Scripts

### `npm run build`

This command compiles the TypeScript code into JavaScript.

### `npm start`

This command starts the application. It first builds the project and then starts the server.

## Environment Variables

The application requires a `.env` file with the following variables:
- `NODE_ENV`
- `SERVICE_NAME`
- `DB_HOST`
- `DB_PORT`
- `DB_USER`
- `DB_PASSWORD`
- `DB_NAME`
